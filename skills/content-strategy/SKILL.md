# コンテンツ戦略スキル

蓄積された動画パフォーマンス・コメントデータを元に、データドリブンなコンテンツ戦略を策定する。

## いつ使うか

- 次に作るべき動画のテーマを決めたいとき
- 投稿スケジュールを最適化したいとき
- どんなコンテンツが伸びるか予測したいとき
- 動画企画のアイデアをデータから発見したいとき

## いつ使わないか

- 既存動画の分析のみ → video-performance を使う
- コメントの深掘り分析 → comment-analysis を使う

## 手順

### Step 1: データ取得

**高パフォーマンス動画の特徴分析**:
```
get_videos(channel_id=対象, limit=100, sort_by=views_total, sort_order=desc, include_channel=true)
```

**視聴者リクエストの収集**:
```
get_comments(channel_id=対象, query=してほしい, limit=50, sort_by=like_count, sort_order=desc)
get_comments(channel_id=対象, query=知りたい, limit=50, sort_by=like_count, sort_order=desc)
get_comments(channel_id=対象, query=お願い, limit=50, sort_by=like_count, sort_order=desc)
```

**投稿パターンの分析**:
```
get_videos(channel_id=対象, limit=100, sort_by=published_at, sort_order=desc)
```

**直近の成長トレンド**:
```
get_snapshots(channel_id=対象, start_date=90日前, granularity=weekly, include_deltas=true)
```

### Step 2: 分析

#### 2-1. 勝ちパターンの抽出

上位 20% の動画に共通する要素を特定:

- **テーマ**: どのジャンル・話題が伸びているか
- **タイトル構造**: 数字、疑問形、How-to 等
- **動画尺**: 最適な長さ
- **公開タイミング**: 曜日・時間帯
- **タグ**: 効果的なキーワード

#### 2-2. ギャップ分析

視聴者が求めているが、まだ作っていないコンテンツを特定:

1. コメントから抽出したリクエスト一覧
2. 高いいねのリクエスト = 需要が確認されているテーマ
3. 既存動画とのマッチング → 未カバーのテーマを特定

#### 2-3. コンテンツミックス最適化

現在のコンテンツ構成を分析し、最適なバランスを提案:

| コンテンツタイプ | 役割 |
|---|---|
| ヒーロー型 | 大きなリーチを狙う (月1-2本) |
| ハブ型 | 定期的な視聴者を維持 (週1-2本) |
| ヘルプ型 | 検索流入を稼ぐ (常設) |
| ショート | 新規視聴者の獲得 |

#### 2-4. 投稿スケジュール分析

過去の投稿データから:
- 最も再生数が伸びやすい曜日
- 投稿頻度と再生数の相関
- 連続投稿 vs 間隔を空けた投稿の効果

### Step 3: 出力

```markdown
## コンテンツ戦略レポート: [チャンネル名]

### サマリー
- 分析基盤: [動画数]本のパフォーマンスデータ + [コメント数]件の視聴者フィードバック
- 戦略の方向性: [一言で]

### 勝ちパターン
| 要素 | 上位動画の特徴 | 下位動画の特徴 |
|---|---|---|
| テーマ | ... | ... |
| タイトル | ... | ... |
| 尺 | ... | ... |
| 投稿日 | ... | ... |

### 企画案 TOP 5
| 優先度 | 企画タイトル案 | 根拠 | 想定再生数 |
|---|---|---|---|
| 1 | ... | [データ根拠] | ... |
| 2 | ... | [データ根拠] | ... |

### 視聴者が求めているテーマ
[コメントから抽出したリクエスト + いいね数]

### 投稿スケジュール推奨
- 最適な投稿曜日: [曜日] (根拠: ...)
- 推奨投稿頻度: [週X本] (根拠: ...)
- コンテンツミックス: ヒーロー[X]本 / ハブ[X]本 / ヘルプ[X]本 / ショート[X]本 (月間)

### 3ヶ月アクションプラン
#### 月1: [テーマ]
- ...
#### 月2: [テーマ]
- ...
#### 月3: [テーマ]
- ...
```

## 注意事項

- 過去データから未来を完全に予測することはできない。提案はあくまで「確率が高い方向性」
- 視聴者リクエストが多い = 必ず伸びるとは限らない。既存視聴者と新規視聴者の需要は異なる
- 想定再生数は過去の類似動画のパフォーマンスから推定した目安。保証値ではない
- トレンドは変化するため、この戦略は3ヶ月ごとに見直す
