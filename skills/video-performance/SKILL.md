# 動画パフォーマンス分析スキル

動画の数値指標を多角的に分析し、成功要因・改善ポイントを特定する。

## いつ使うか

- 特定の動画がなぜ伸びた/伸びなかったか知りたいとき
- 動画のパフォーマンスを評価・ランキングしたいとき
- 再生数の伸び方のパターンを分析したいとき
- エンゲージメント率の改善策を探したいとき

## いつ使わないか

- コメントの内容分析 → comment-analysis を使う
- チャンネル間の比較 → channel-comparison を使う

## 手順

### Step 1: データ取得

**動画一覧 + 集計統計**:
```
get_videos(channel_id=対象チャンネルID, limit=50, sort_by=views_total, sort_order=desc, include_channel=true)
```

**個別動画の詳細 + スナップショット**:
```
get_video(video_id=対象動画ID, include_snapshots=true)
```

**時系列データ**:
```
get_snapshots(channel_id=対象チャンネルID, start_date=30日前, end_date=今日, granularity=daily, include_deltas=true)
```

取得時のポイント:
- `stats` フィールドの `views_median`, `engagement_rate_avg` をベンチマークとして使う
- `views_per_day` で公開からの1日あたり再生数を比較すると、公開時期の差を吸収できる
- `is_short` フィルタでショート動画と通常動画を分けて分析する

### Step 2: 分析

#### 2-1. パフォーマンス評価マトリクス

各動画を以下の指標で評価する:

| 指標 | 計算方法 | 見方 |
|---|---|---|
| 再生数 | `views_total` | チャンネル中央値との比較 |
| エンゲージメント率 | `(likes + comments) / views` | 3%以上は高い |
| いいね率 | `likes / views` | 視聴者の共感度 |
| コメント率 | `comments / views` | 視聴者の参加度 |
| 日次再生数 | `views / 公開日からの日数` | 時期を正規化した比較 |

#### 2-2. タイトル・サムネイル分析

上位/下位動画のタイトルを比較し、パターンを抽出:

- タイトル文字数
- 数字の使用 (「3つの方法」等)
- 疑問形 vs 断定形
- キーワードの傾向
- 公開曜日・時間帯

#### 2-3. 成長パターン分析

スナップショットデータから再生数の伸びを分類:

| パターン | 特徴 | 意味 |
|---|---|---|
| 初速型 | 公開後3日で急伸、その後停滞 | サムネイル/タイトルは強いが検索流入が弱い |
| 安定成長型 | 緩やかに継続して伸びる | SEO が効いている、evergreen コンテンツ |
| 遅咲き型 | 公開後しばらくして急伸 | 外部からの流入、アルゴリズム推奨 |
| 低空飛行型 | 公開後も伸びない | タイトル/サムネイル/テーマの改善が必要 |

#### 2-4. ショート vs 通常動画

ショート動画と通常動画を分けて以下を比較:
- 平均再生数の違い
- エンゲージメント率の違い
- 投稿頻度と再生数の関係

#### 2-5. ベンチマーク比較

チャンネル全体の統計 (`stats`) と個別動画を比較:
- 中央値 (`views_median`) を超えているか
- 上位 10% に入る動画の共通点
- 下位 10% の動画の共通点

### Step 3: 出力

```markdown
## 動画パフォーマンスレポート: [チャンネル名 or 動画タイトル]

### サマリー
- 分析対象: [動画数]本 (期間: [開始日] 〜 [終了日])
- 平均再生数: [X] / 中央値: [Y]
- 平均エンゲージメント率: [X]%
- 総合評価: [一言で]

### パフォーマンスランキング
| 順位 | 動画タイトル | 再生数 | エンゲージメント率 | 日次再生数 | 評価 |
|---|---|---|---|---|---|

### 成功要因分析
#### 上位動画の共通点
- [発見 1]
- [発見 2]

#### 下位動画の共通点
- [発見 1]
- [発見 2]

### 成長パターン
[該当動画のスナップショットデータに基づく分析]

### ショート vs 通常動画
| 指標 | ショート | 通常 |
|---|---|---|

### 推奨アクション
1. [タイトル改善案 + 根拠]
2. [投稿タイミング最適化 + 根拠]
3. [コンテンツテーマの提案 + 根拠]
```

## 注意事項

- 再生数だけで動画の価値を判断しない。エンゲージメント率が高い動画はファン化に貢献している
- 公開日が新しい動画は再生数が少なくて当然。`views_per_day` で正規化して比較する
- ショート動画と通常動画は視聴者の行動が異なるため、必ず分けて分析する
- 外部要因 (トレンド、コラボ、広告) による再生数の増加は、コンテンツ自体の評価とは分けて考える
