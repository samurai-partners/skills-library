# スキル名

## 概要

このスキルの目的と提供する価値を1-2文で説明する。

## いつ使うか

- ユーザーが「〇〇を分析して」と言ったとき
- 〇〇のデータを元にレポートが必要なとき

## いつ使わないか

- 〇〇ではないデータの分析
- リアルタイムデータが必要な場合

## 手順

### Step 1: データ取得

MCP Server から必要なデータを取得する。

```
使用する MCP tool:
- get_videos: 動画一覧を取得
- get_comments: コメントを取得
```

取得時のパラメータ指針:
- limit: 分析に必要な件数 (通常 50-100)
- sort_by: 分析目的に応じたソート

### Step 2: 分析

取得したデータに対して以下の観点で分析する:

1. **観点 1**: 〇〇
2. **観点 2**: △△
3. **観点 3**: □□

### Step 3: 出力

以下のフォーマットでレポートを出力する:

```markdown
## 分析レポート: [タイトル]

### サマリー
- 結論を1-2文で

### 詳細分析
#### 観点 1
- 発見事項
- データ根拠

### 推奨アクション
1. 具体的なアクション
2. 優先度付き
```
